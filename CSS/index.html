1 . What is CSS ?

Cascading Style Sheets (CSS) is a style sheet language used for explaining the look and display of a document any written in a markup language


2 . CSS Syntax :

  1. selector - (selector name should not start with alphabets) points to the HTML element you we to style
  2. declaration - Each declaration includes a CSS property name and a value, separated by a colon Inside the curly braces separated by semicolons,declaration always ends with a semicolon
  3. selectors : class, id ,attribute,value,chaild selectors,element , name, All (*) selector
    • property name - name of the style
    • property value - value of the style

3 . Ways (4) to add CSS to HTML :

* we layedown 4 types in a browser priority order bellow

  1. Inline css
  2. Internal css
  3. External css
  4. Imported css
1 . Inline css :
we can use style attribute of any HTML element to define style rules. These rules will be applied to that element only

2 . Internal css :
we can put our CSS rules inside HTML document using the style element. This tag is placed inside head tags

3 . External css :
The link element can be used to include an external stylesheet file in your HTML document. An external style sheet is a separate text file with .css extension. You define all the Style rules within this text file and then you can include this file in any HTML document using link element.

4 . Imported css :
@import is used to import an external stylesheet in a manner similar to the link element. Here we place it inside head tag again

8 . Playing with Position and float :

A . Position :

The position property defines the type of positioning methodused for an document element.
properties will not work unless the position property is set & works depending on the position value.
we have 5 position types

  1. Static

    2. HTML elements are positioned static by default.they will not affected by the top, bottom, left, and right properties.they positioned according to the normal flow of the page:

    static


  2. Relative

    3. An element with position: relative; is positioned relative to its normal position.
    Setting the top, right, bottom, and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.

    relative(top: 40px;)


  3. Absolute

    4. An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed).
    However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling

    absolute(top: 0;right: 0;)


  4. Sticky

    5. An element with position: sticky; is positioned based on the user's scroll position. A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).

    it sticks to bottom sticky element(bottom:0px;) , scroll to bottom to see effect


  5. Fixed

    6. See top most & Right corner of window

    An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.
    A fixed element does not leave a gap in the page where it would normally have been located.

    fixed positioned division


B . Float :

The float property specifies how an element should float.
The float property is used for positioning and layout on web pages.

  1. left - The element floats to the left of its container
  2. right - The element floats to the right of its container
  3. none - The element does not float (will be displayed just where it occurs in the text). This is default
  4. inherit - The element inherits the float value of its parent

11.css overflow :

The overflow property specifies what happens if content overflows an element's box.This property specifies whether to clip content or to add scrollbars when an element's content is too big to fit in a specified area.


NOTE:The overflow property only works for block elements with a specified height.

The css overflow contains the following properties.

  1. Visible
  2. Hidden
  3. Scroll
  4. Auto
  5. Overflow-x
  6. overflow-y

overflow:visible

Content is not clipped and may be rendered outside the content box. Default value.Content is not clipped and may be rendered outside the content box. Default value.

overflow:hidden

Content is clipped if necessary to fit the content box. No scrollbars are provided.
