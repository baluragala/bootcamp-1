Display


Block-level Elements:

Block level element starts on a new line and it occupy full width.

  • <div>
  • <h1> - <h6>
  • <p>
  • <form>
  • <header>
  • <footer>
  • <section>

Example:

h3.heading

div.heading

p.paragraph



Inline Elements:

An inline element does not starts on a new line and it takes necessary width.

  • <span>
  • <a>
  • <img>

Example:

span.heading a.link




Display Value Override:

  • 1.HTML
  • 2.CSS
  • 3.JavaScript



display:none;

Hides an element and the page will be displayed
as if the element is not there.

hidden heading
