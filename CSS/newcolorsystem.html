Css Colors

RGBA Colors:

RGBA means Red,Green,Blue,Alpha(alpha parameter a number between 0.0 to 1.0).

Example:

rgba(255, 50, 0, 0.2);
rgba(255, 0, 50, 0.4);
rgba(255, 40, 0, 0.6);
rgba(255, 80, 0, 0.8);

HSL Colors:

HSL means Hue(degree(0 to 360)),Saturation(percentage) and Lightness(percentage).

Example:

hsl(20, 100%, 30%);
hsl(30, 100%, 50%);
hsl(40, 100%, 70%);
hsl(50, 100%, 90%);

HSLA Colors:

HSLA means Hue(degree(0 to 360)),Saturation(percentage),Lightness(percentage) and Alpha.

Example:

hsla(0, 100%, 30%, 0.3);
hsla(20, 100%, 40%, 0.3);
hsla(0, 100%, 50%, 0.3);
hsla(40, 100%, 60%, 0.3);
